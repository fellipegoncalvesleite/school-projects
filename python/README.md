# Special Topics in IT

Class activities from the Special Topics in IT course, written in Python.

Activities are organized by class date under `src/` (e.g. `src/16.05`,
`src/08.06`), using numpy and matplotlib.

## How to run

The project uses [uv](https://docs.astral.sh/uv/):

```bash
uv sync
uv run src/<date>/activity-01.py
```
