# app_gerenciamento

Flutter app built for the 2nd term graded assignment.

## Goal

The project shows a login screen for a personal task management app.

## What the screen has

- Email field.
- Password field.
- Sign in button.
- Forgot my password link.
- Create account link.
- Icon representing the app.

## Folder structure

```text
lib/
  main.dart
  src/
    app.dart
    ui/
      view_models/
        home_view_model.dart
      views/
        home/
          home_page.dart
          widgets/
            app_logo.dart
            link_button.dart
            login_field.dart
            primary_button.dart
```

## How to run

```bash
flutter run
```
