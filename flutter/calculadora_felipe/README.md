# calculadora_felipe

Flutter app built for the mobile development classes.

## Goal

The project implements a simple calculator for basic operations.

## Features

- Addition, subtraction, multiplication and division.
- Percentage.
- Decimal numbers (comma separator).
- C button clears everything, < deletes the last digit.
- The display shows the expression while the numbers are typed.

## Folder structure

```text
lib/

  src/

    ui/
      view_models/
        home_view_model.dart
      views/
        home/

          widgets/

```

## How to run

```bash
flutter run
```
